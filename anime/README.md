# PROJETO ANIMES

  Esse projeto consiste na busca por dados na internet com uso do Selenium e BS4 do Python,
alocar esses dados em um banco de dados qualquer para fazer uma análise descritiva utilizando SQL e Python,
após isso é feito um dashboard utilizando Power BI e Python (talvez) e, por fim, é feito um projeto de ciência de dados para
catalogação dos animes de acordo os dados obtidos e informados.
