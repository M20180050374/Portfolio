# Portfolio
Portfolio com os projetos pessoais. 

O intuito é demonstrar de forma prática meus conhecimentos com exemplos que podem ser utilizados em qualquer ambiente profissional ou pessoal.

## ANIME

Esse projeto consiste na busca por dados na internet sobre animes, tratamento desses dados, criação de um dashboard e uma classificação dos animes para recomendação de acordo com os dados sobre as 'qualidades' dos animes. Acho interessante? Acompanha o detalhamento de cada etapa.
